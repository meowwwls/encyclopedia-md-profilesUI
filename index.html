Biography

Hi folks. I’m Paul. I’m a front-end developer who loves the web. I’m always thinking a lot about how to make you more productive with tools that improve your workflow, help you make slicker, sexier, mobile web sites and webapps.

I work on the Google Chrome team focused on Developer Tooling. Chrome DevTools and everything else Chrome can do to make you a productive web developer.

I’ve developed tools like Modernizr, Yeoman, CSS3 Please and HTML5 Boilerplate, curated HTML5 Rocks, and co-created projects like HTML5 Please, mothereffingtextshadow, mothereffinghsl, Move the Web Forward, Mother-effing Animated GIF, W3Fools, chromestatus.com, HTML5 Readiness, Font-end Coding Standards, Type Rendering Project.

Trivia

I used to host the amazing yayQuery Podcast. I’ve long loved music and kept the Aurgasm music blog up for 10 years.
